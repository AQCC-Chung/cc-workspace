## 2025-05-18 - Tooltip ARIA Labels in FitTracker
**Learning:** Adding ARIA labels to icon-only buttons drastically improves accessibility for screen reader users and improves general semantic HTML structure. Icon-only buttons without context are an extremely common failure point in modern React components, as shown in the `WeightTraining` and `CalendarView` components where `◀`, `▶`, `⚙️`, and `✕` characters were used standalone. When adding these, it's critical to match the application's existing localization language (in this case, Traditional Chinese: "設定", "上個月", "下個月", "關閉").
**Action:** Audit all icon-only interactive elements in new components and always enforce the inclusion of localized `aria-label`s. Include a Playwright UI test to programmatically verify visibility based on these aria-labels.
